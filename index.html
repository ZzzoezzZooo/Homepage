Selected Projects

  • Mini-OS Migration on Smarpthone for IoT Applications 2015
    Proposed a new migration solution that enables a Mini-OS to migrate between ARM based devices. We implemented a prototype XEN-MiniOS on exynos 5250 development board, and the evaluation shows that our solution is practical and efficient.

  • Detecting data leakage via Covert Channels on Smartphone 2015
    Implemented an enhanced TaintDroid system that can track data transferring via five types of covert channels (e.g. Speaker ! Microphone. Speaker ! Accelerometer). We implemented this system on galaxy nexus and nexus 10.

  • Record and Replay Tool for Android Applications 2014-2015

    Proposed a novel tool that records and replays all input to an application without requiring root privilege of the mobile device. This tool adopts a client-server architecture, in which the to-be-tested Android applications are executed in a remote server while a client is running on the local smart device. Specifically, input (sensor, gps, user actions etc.) is recorded in the client and forwarded to the application running on the remote server; output (screen content) is sent back to the client in real time, and then rendered on the device screen so that the user has the same experience as if the application is running locally. The tool has been evaluated with tens of popular mobile applications, showing that it is efficient, flexible, and comprehensive.

  • A secure Runtime Environment for Android Applications 2013-2014
    Proposed a system that protects the runtime environment for sensitive Android applications even when the operating system is compromised. This system combines KVM for ARM technique and clientserver architecture, protecting not only the memory but also the input/output of sensitive applications. We implemented and evaluated a prototype system on chromebook.

  • Port VxWorks OS to MIPS-Based Development Boards 2008-2012
    Ported VxWorks to the latest Loogson products (MIPS-based). This work includes porting drivers for network card, USB device, hard drive, graphics, audio, keyboard, mouse, serial port, and parallel port, and rewriting the VxWorks core such that it can support new features in the Loogson products.
